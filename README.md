# ThinkingCode Education

A collection of full-stack projects built during **ThinkingCode Education** seminars. Each project includes the prompts, specifications, and folder scaffolding used to generate production-grade applications with AI-assisted development.

## Structure

```
projects/
└── <project-name>/
    ├── CLAUDE.md             # Project overview & conventions
    ├── PROMPT-BACKEND.md     # Backend generation prompt
    ├── PROMPT-FRONTEND.md    # Frontend generation prompt
    ├── backend/              # Backend folder scaffold
    └── frontend/             # Frontend folder scaffold
```

## Projects

| Project | Description |
|---------|-------------|
| [FolioForge](projects/folioforge/) | A portfolio generation platform — users complete a guided onboarding wizard (or paste resume text) and the system generates a shareable portfolio page with PDF download. Express.js API + Next.js 14 frontend. |

## How to Use

1. Browse into a project folder.
2. Read the `PROMPT-BACKEND.md` and `PROMPT-FRONTEND.md` — these are the full specifications used to generate the application source code.
3. The directory structure under `backend/` and `frontend/` shows the intended project layout.
4. Use the prompts with an AI coding assistant (e.g. Claude Code) to generate the complete application.
